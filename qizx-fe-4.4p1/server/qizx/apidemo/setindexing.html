setindexing

Define or redefine the Indexing Specification of the Library.



HTTP Request: POST

Parameters:

NameValueDescription
indexingan XML document containing the Indexing Specification.
libraryName of Library used (can be empty if exactly one Library in server).

Response:

   mime-type: text/plain

The name of the Library.

 

Error codes:

XMLData
if the uploaded specification is invalid.
AccessControl
if the user does not have the role allowing administration operations.
