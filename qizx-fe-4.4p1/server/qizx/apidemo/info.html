info

Get information about the server and the XQuery engine.

The output format is similar to properties, with a top element "info":

<info> <property
    name="server-name">Qizx Server</property> <property
    name="product-name">Qizx</property> <property
    name="product-version">4.0</property> ... </info>

HTTP Request: GET

Parameters: none

Response:

 mime-type: text/xml

A list of properties.

 

Error codes:

Server
problem on the server (offline).
