getindexing

Get the Indexing Specification of the Library as an XML document.



HTTP Request: GET

Parameters:

NameValueDescription
libraryName of Library used (can be empty if exactly one Library in server).

Response:

   mime-type: text/xml

The Indexing Specification of the Library in XML.

 

Error codes:

AccessControl
if the user does not have the role allowing administration operations (except for 'status' which is allowed to all users).
