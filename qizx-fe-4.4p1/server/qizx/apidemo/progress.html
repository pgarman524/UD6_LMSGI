progress

Return progress information about a long task such as backup, commit etc.

Â 

HTTP Request: GET

Parameters:

Name Value Description
id progress identifier returned by the monitored request.

Response:

mime-type: text/plain

Plain text on 2 lines:

  • task name + library name

  • percentage done, as a decimal number in range 0 to 1.

Example:

backup xlib 
0.93

Error codes:

NotFound
when the progress id does not exist.
