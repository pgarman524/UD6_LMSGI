Contents of the distribution

See manual for more details.

docs
Documentation. Contains both PDF and HTML versions of the manual.
bin
Contains launcher scripts for Qizx applications: (.bat extension on Windows). These applications are documented in the manual.
lib
Main library: qizx.jar.
Visual interface Qizx Studio: qizxstudio.jar
legal
Licenses and readme for Qizx and third-party libraries.
server
Material for building a server instance.
src.zip
Source code of the applications, API and API utility classes.
